OverEasy

Getting Started

Install

Install OverEasy via NPM

npm install overeasy

This will give you compiled CSS, along with all the source files to customize your settings or optimize your build.

If you're just after the CSS, you can use the direct CDN link

<link href="https://unpkg.com/overeasy/dist/css/overeasy.min.css" rel="stylesheet">

If you prefer, you can clone the GitHub repo (or view it here)

git clone https://github.com/overeasy-css/overeasy.git

Usage

OverEasy is a collection of modular CSS style utilities, most of which perform just one function at a time. This makes it easy to mix and match tools to quickly achieve your outcome inline. There are also some common settings and styles to get the basics up and running quickly, all configurable from one place. The utilities and settings are organized into functional Modules that are easy to customize or omit to suit your needs and it comes packaged with everything you need to modify and recompile quickly and easily.

  • Whitespace - Atomic margin and padding utilities
  • Layout - Content container, responsive states, and display modes
  • Grid - Responsive column grid with customizable number of columns
  • Flex - Flex box parent and child display utilities
  • Typography - Body and Heading text, alignment, lists, tables
  • Colors - Auto-generated array of color utilities easily modified to any palette
  • Backgrounds - Positioning, size, repeat controls and palette color utilities
  • Borders - Weight, rounding, colors
  • Buttons - Default styling compatible with other utilities
  • Input Fields - Form fields and input styling for quick pretty
